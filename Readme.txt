# Face Recognition Project

This project involves building face recognition system using Python, OpenCV, and face_recognition library. The system captures a photo of a person through the webcam or existing photo, processes it, and compares it with a collection of images in an album to recognize and identify the person.
Basically it helps an Individual to find his/her photo from an album.

## Prerequisites

Before running the project, make sure you have the following installed:

- Python 3.8.10
- OpenCV
- face_recognition library
-dlib librabry
-cmake library
- (Any other dependencies)
